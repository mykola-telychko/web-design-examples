1. Time Grid Scheduler (Сітковий планувальник)
Планувальник, який використовує сітку для відображення часу та подій.

Приклад: Google Calendar, Outlook Calendar.

2. Weekly Planner (Тижневий планувальник)
Планувальник, який відображає події протягом тижня.

Приклад: Trello (у вигляді дошки), Asana.

3. Daily Scheduler (Щоденний планувальник)
Планувальник, який фокусується на подіях одного дня.

Приклад: Todoist, Any.do.

4. Calendar View (Перегляд календаря)
Загальний термін для інтерфейсів, які відображають події у вигляді календаря.

Приклад: Apple Calendar, Fantastical.

5. Resource Scheduler (Планувальник ресурсів)
Використовується для планування ресурсів (наприклад, кімнат, обладнання, людей).

Приклад: Microsoft Bookings, Calendly.

6. Gantt Chart (Діаграма Ганта)
Планувальник, який відображає завдання та їх тривалість на часовій шкалі.

Приклад: Monday.com, Smartsheet.

7. Timeline View (Перегляд часової шкали)
Планувальник, який відображає події на горизонтальній часовій шкалі.

Приклад: Notion (Timeline View), Airtable.

8. Event Scheduler (Планувальник подій)
Використовується для планування конкретних подій або заходів.

Приклад: Eventbrite, Doodle.

9. Booking System (Система бронювання)
Планувальник, який дозволяє бронювати час або ресурси.

Приклад: Calendly, Acuity Scheduling.

10. Time Blocking Planner (Планувальник з блокуванням часу)
Планувальник, який дозволяє розподіляти час на блоки для різних завдань.

Приклад: Clockwise, SkedPal.

11. Shift Planner (Планувальник змін)
Використовується для планування робочих змін.

Приклад: When I Work, Shiftboard.

12. Agenda View (Перегляд порядку денного)
Планувальник, який відображає події у вигляді списку.

Приклад: Google Agenda, Outlook Agenda.

13. Kanban Board with Timeline (Канбан-дошка з часовою шкалою)
Поєднання канбан-дошки з елементами планування часу.

Приклад: ClickUp, Monday.com.

14. Meeting Scheduler (Планувальник зустрічей)
Спеціалізований планувальник для організації зустрічей.

Приклад: Doodle, Calendly.

15. Project Timeline (Таймлайн проекту)
Планувальник, який відображає етапи проекту на часовій шкалі.

Приклад: Asana Timeline, Microsoft Project.

16. Interactive Scheduler (Інтерактивний планувальник)
Планувальник, який дозволяє користувачам взаємодіяти з подіями (перетягувати, змінювати час тощо).

Приклад: FullCalendar (бібліотека JavaScript), Scheduler.js.

17. Visual Scheduler (Візуальний планувальник)
Планувальник, який використовує візуальні елементи для відображення подій.

Приклад: TimeTree, Teamup.

18. Employee Roster (Розклад роботи співробітників)
Планувальник для управління робочими графіками співробітників.

Приклад: Deputy, Homebase.

19. Class Scheduler (Розклад занять)
Планувальник для навчальних закладів або курсів.

Приклад: Google Classroom, Canvas.

20. Resource Calendar (Календар ресурсів)
Планувальник, який відображає доступність ресурсів (кімнати, обладнання тощо).

Приклад: Roomzilla, Skedda.

Де шукати референси:
Google Calendar (класичний приклад сіткового планувальника).

Trello (канбан-дошка з можливістю додавання дедлайнів).

Notion (Timeline View для планування проектів).

FullCalendar (бібліотека для створення календарів).

Monday.com (планувальник для команд і проектів).

Asana (Timeline View для управління проектами).

Calendly (планувальник для бронювання зустрічей).

Fantastical (календар з гарним дизайном та функціональністю).
